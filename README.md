# Project Universe ✨

**Project Universe** is a high-fidelity, cosmic-themed Android notes application designed with a stunning "Frosted Glass" (Glassmorphic) aesthetic. It combines advanced security, AI-powered productivity, and persistent local storage into a seamless user experience.

---

## 🚀 Key Features

- **🌌 Cosmic UI**: A beautiful, translucent glassmorphic interface with staggered masonry layouts and dynamic light-refraction card edges.
- **🔒 Biometric Security**: Secure your "Secret Universe" using integrated Fingerprint/Face authentication. Notes marked as secret remain hidden until verified.
- **📷 AI Text Scanning (OCR)**: Instantly transcribe physical documents into digital notes using Google ML Kit's on-device text recognition.
- **💾 Persistent Storage**: All your notes are saved locally using **Room Database**, ensuring your data is safe even after app restarts.
- **🎨 Personalization**: Organize notes with cosmic color palettes and pin your most important thoughts to the top.
- **📅 Deadlines**: Set reminders and track deadlines for your notes with built-in date and time pickers.

---

## 🛠️ Tech Stack

- **Language**: Pure Java
- **Database**: Room Persistence Library (SQLite)
- **Security**: AndroidX Biometric Library
- **AI/ML**: Google ML Kit (Text Recognition)
- **UI Components**: 
  - Material Design 3
  - StaggeredGridLayoutManager
  - ViewPropertyAnimator for smooth transitions
  - Custom Layer-Lists for Glassmorphic effects

## ⚙️ Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Vedanshdhingra/project.git
   ```

2. **Open in Android Studio**:
   - File > Open > Select the cloned folder.

3. **Build & Run**:
   - Sync Gradle files.
   - Run on an Android device (API 24+).
   - *Note: Camera and Biometric hardware are required for full feature testing.*
